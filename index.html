I
Unit I
Introduction to AI & Problem Solving
State Space Search Problem Formulation AI Techniques Toy Problems
This app demonstrates the fundamental AI concept of state space search — formulating a problem as a grid (state space) with a start and goal state, then finding a solution path through search. This directly applies the Problem Solving Process from Unit I.
II
Unit II
Data Structures & Search Algorithms
BFS DFS UCS A* Algorithm Heuristic Search Queues / Stacks
The PATH SEARCH tab implements all Unit II algorithms. BFS uses a queue (FIFO), DFS uses a stack (LIFO), UCS uses a priority queue on cost, and A* uses Manhattan distance as a heuristic — combining cost and heuristic (informed search).
III
Unit III
Adversarial Search & Intelligent Agents
Minimax Alpha-Beta Pruning Game Tree Rational Agent MAX / MIN Players
The MINIMAX tab implements an Intelligent Agent (the AI player) using the Minimax algorithm with Alpha-Beta Pruning. The AI acts as a rational agent — it perceives the board state, applies DFS on the game tree, and selects the optimal move, exactly as taught in Unit III.
